# Creative arsenal (advanced inspiration)

Consult when you need differentiated patterns beyond baseline cards and hero clichés.

**Important:** Prefer **Framer Motion** for component-level UI choreography. Reserve **GSAP** or **Three.js / WebGL** for isolated canvas / scroll narratives with full cleanup—and **never** combine GSAP/Three with Framer in the **same component tree**.

## Hero paradigm

Reject “center paragraph on moody photo.” Explore asymmetric halves, purposeful image fades into background (light/dark aware), and editorial alignment.

## Navigation and menus

- macOS dock magnification along an edge
- Magnetic buttons (pointer via motion values, not React state thrash)
- Gooey morphing submenus
- Dynamic Island status surfaces
- Contextual radial menus anchored to click coordinates
- Floating speed-dial arcs
- Mega menus with staggered reveals

## Layout and grids

- Bento / control-center tiles with irregular spans
- Masonry waterfalls
- Chroma grids (subtile animated gradients on rails)
- Split-screen opposing scroll velocities
- Curtain hero reveals splitting vertically

## Cards and surfaces

- Parallax tilt read from pointer data
- Spotlight borders tracking cursor
- True glass panels with inner-edge refraction
- Iridescent “foil” treatments (sparingly, on-brand)
- Tinder-style swipe stacks
- Morph-from-trigger modals expanding into dialogs

## Scroll-led storytelling

- Sticky stacking cards layering over each other on scroll
- Horizontal hijack corridors (ethical, accessible fallbacks!)
- Sequence-driven media tied to scrollbar progress with performance budgets
- Zoom parallax on hero canvases
- SVG progress paths animating strokes with scroll deltas
- Liquid wipe transitions across route changes

## Galleries and media

- Dome / panoramic galleries when WebGL-approved
- Coverflow rails with focal depth cues
- Draggable infinite canvases
- Accordion media strips enlarging on hover/focus
- Hover image trails anchored to motion-safe layers
- Glitch shaders used once, responsibly

## Typography and kinetic type

- Marquees with directional shifts tied to inertia cues
- Text masks revealing looping video responsibly (prefers-reduced-motion)
- Scramblers / decoders for fleeting moments—not entire paragraphs
- Circular text rigs
- Gradient stroke shaders on headlines (sparingly)
- Typography grids reacting to proximity

## Micro-interactions and effects

- Particle burst CTAs sparingly behind reduced-motion guards
- Pull-to-refresh liquids on mobile prototypes
- Directional fills from pointer vector entry
- Ripples seeded from precise click coordinates (also provide non-motion affordance)
- Animated SVG contours for iconography flair
- Mesh gradients as ambient backgrounds—not competing with text
- Depth-of-field focus pulls for contextual modals

Use this inventory as inspiration, never as a dumping ground—everything must reconcile with SKILL.md dial settings, forbidden tells, accessibility, and performance budgets.
